## One-line Description

ArcReady is an open-source Arc-specific CI quality gate and integration validator for wallets, bridges, App Kit integrations, and dApps.

## Short Description

ArcReady helps Arc builders catch integration mistakes before they ship.

It validates common Arc-specific assumptions around wallet UX, bridge flows, App Kit integrations, USDC gas, finality, CCTP, and Unified Balance UX.

The MVP includes a TypeScript CLI, rule packs, multiple report formats, GitHub Action support, demo fixtures, and 114 passing tests.

## Technical Positioning

```text
ArcReady is not a generic multi-chain linter.

It is focused on Arc-specific integration assumptions and CI validation for wallets, bridge flows, App Kit integrations, and dApps.
```

## What To Emphasize

* Arc-specific validation
* CI quality gate
* Local-first
* Open-source-first

## What Not To Claim

Do not claim:

* ArcReady is an official Arc or Circle product
* ArcReady certifies integrations
* ArcReady is production hardened
* ArcReady is a security audit tool
* ArcReady is published on npm
* ArcReady is available on GitHub Marketplace
* ArcReady supports every Arc integration pattern
