# Codex Tasks

## Current Foundation

- Create the pnpm workspace.
- Add the `packages/arcready` CLI package.
- Implement `arcready init` and `arcready scan`.
- Return a stub report from `arcready scan`.
- Add placeholder folders for future Arc-specific rules, presets, reporters, config, findings, and scoring.
- Add basic tests for CLI boot and scan report generation.

## Explicit Non-Goals

- No web app.
- No database.
- No auth.
- No telemetry.
- No hosted dashboard.
- No runtime RPC calls.
- No real rule implementation yet.
